jquery post 在firefox浏览器中遇到的问题

发表于 2016-07-21 | 分类于 前端

我在页面中使用按钮发送表单数据，同时还调用$.post发送请求， $.post(“/login/create”, info, function (data, textStatus) { console.log(data); if (data.ret == ‘SUCCESS’) { $.cookie(…); $.cookie(…); } else { alert(data.ret); } }, ‘json’); 在chrome内核浏览器和ie浏览器下显示正常，但是在firefox中不能接收$.post传回的数据。 解决办法：按钮默认属性是submit，提交和$.post同时进行的话在firefox中会出错。将提交按钮类型改为button，然后在$.post中添加document.getElementById(“form”).submit();即可解决。
